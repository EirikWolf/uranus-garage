# P3 Moonshot Backlog

Disse ideene krever videre utforskning og planlegging.

## 1. Brew Intelligence — Prediktiv Bryggeanalyse
- Bruk aggregert bryggedata til å predikere utfall
- "Basert på din oppskrift, forventer vi OG 1.052 ±0.003"
- Automatisk varsling ved avvik i gravity-kurve
- Krever: Vektordatabase, RAG, eller fine-tunet modell

## 2. Social Brewing Protocol — Desentralisert Bryggenettverk
- Åpent API for andre hjemmebryggerier
- Federation av oppskrifter på tvers av plattformer
- Brew credits for kvalitetsbidrag
- Krever: API-design, auth for tredjepart, governance-modell

## 3. Brew Twin — Digital Tvilling
- 3D-simulering av gjæringsprosessen (WebGL)
- "Hva om"-eksperimenter uten å risikere brygget
- Smaksprediksjoner basert på fermentasjonskurve
- Krever: Fysikkbasert gjæringsmodell, WebGL-erfaring, RAPT-integrasjon
